

เว็บไซต์ขายของออนไลน์ที่สมบูรณ์สร้างด้วย Flask พร้อมฟีเจอร์ครบครัน


## 📦 ติดตั้ง

### ข้อกำหนด
- Python 3.8+
- pip

### ขั้นตอนการติดตั้ง

```bash
# 1. สร้าง Virtual Environment
python -m venv venv

# 2. เปิดใช้ Virtual Environment
# บน Windows:
venv\Scripts\activate
# บน macOS/Linux:
source venv/bin/activate

# 3. ติดตั้ง Dependencies
pip install -r requirements.txt

# 4. รันแอปพลิเคชัน
python app.py
```
